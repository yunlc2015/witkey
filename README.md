### 📝 介绍
Witkey威客系统是一款全开源的系统，前后端不分离开发，全部100%开源！

---

### 🫧 系统亮点
~~~
1.前后端不分离：使用SpringBoot MVC框架和 Freemarker 模板引区别；
2.系统安全：操作权限控制，操作日志记录；
3.代码规范：代码严格分层、注释齐全、统一错误码；
4.....
~~~
